# ⌨️ Typing Speed Tester

A **Typing Speed Tester** built with **HTML, CSS, JavaScript, and Python** that helps users measure and improve their typing speed.  
It generates random sentences for practice and calculates **WPM, accuracy, and time taken** automatically.

---

## 📌 Overview

This project provides a simple interface where users type a generated sentence.  
While typing, the application tracks performance metrics and displays results once the sentence is completed.

It also includes a **dark theme** for a cleaner and more comfortable UI.

---

## ✨ Features

- 📝 **Random Sentence Generation**  
  Click the **New Sentence** button to load a new sentence for typing practice.

- ⚡ **WPM Calculation**  
  Automatically calculates **Words Per Minute (WPM)** based on typing speed.

- 🎯 **Accuracy Detection**  
  Compares typed characters with the original sentence to calculate typing accuracy.

- ⏱ **Time Tracking**  
  Measures the total time taken to complete the sentence.

- 🌙 **Dark Theme**  
  Clean and modern dark interface for better user experience.

- 💻 **Simple and Lightweight**  
  Runs locally in the browser with minimal setup.

---
